# Pincher_Phantom_X100_ROS_Humble_RVIZ
Este laboratorio implementa el control y visualización del manipulador Phantom X Pincher X100 mediante ROS2 Humble, Python y Dynamixel Workbench. Se desarrolla una interfaz gráfica (HMI) capaz de comandar y monitorear en tiempo real los movimientos articulares del robot, permitiendo interactuar con los controladores AX-12, publicar comandos, suscribirse al estado de las juntas y visualizar el modelo 3D del brazo en RViz.

El proyecto también incorpora cinemática directa, representando el pose cartesiano del TCP (X, Y, Z) a partir de los parámetros DH medidos experimentalmente. Además, la interfaz permite seleccionar y ejecutar poses predefinidas, graficar la configuración del robot y comparar su comportamiento virtual con el manipulador real.
